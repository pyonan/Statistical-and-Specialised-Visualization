## 4. Theory

### 4.1 Introduction to Statistical Visualization

Statistical visualization helps in understanding the underlying structure of data. It is widely used in data science to analyze trends, detect patterns, and make predictions.

---

### 4.2 Area Plot

An area plot is similar to a line chart but fills the area under the curve.

#### Characteristics:
- Shows magnitude of values over categories  
- Useful for comparing multiple variables  
- Provides better visual impact  

---

### 4.3 Pie Chart

A pie chart represents data as portions of a circle.

#### Characteristics:
- Each slice represents a category  
- Shows percentage distribution  
- Useful for proportional comparison  

#### Special Feature:
- **Explode effect** separates slices for emphasis  

---

### 4.4 Donut Chart

A donut chart is a modified pie chart with a hole in the center.

#### Advantages:
- More visually appealing  
- Easier to compare segments  

---

### 4.5 Box Plot

A box plot is used to represent the distribution of data.

#### Key Components:
- Minimum value  
- First quartile (Q1)  
- Median  
- Third quartile (Q3)  
- Maximum value  

#### Importance:
- Detects outliers  
- Shows spread of data  

---

### 4.6 Heatmap (Correlation Matrix)

A heatmap is used to visualize the correlation between variables.

#### Characteristics:
- Uses colors to represent values  
- Helps identify relationships between variables  

#### Correlation Values:
- +1 → Strong positive correlation  
- 0 → No correlation  
- -1 → Strong negative correlation  

---

### 4.7 Bubble Plot

A bubble plot is an advanced scatter plot where the size of each point represents an additional variable.

#### Characteristics:
- X-axis and Y-axis show two variables  
- Bubble size represents third variable  
- Color can represent another variable  

#### Importance:
- Displays multi-dimensional data  
- Helps identify patterns and clusters  

---

### 4.8 Multi-colored Bubble Plot

This is an enhanced version of a bubble plot where:
- Color represents additional information  
- Size represents magnitude  
- Improves visual clarity  

---

### 4.9 Outlier Detection using Box Plot

Outliers are extreme values that differ significantly from other data points.

#### Importance:
- Helps detect anomalies  
- Improves data quality  
- Useful in financial and scientific analysis  

---

### 4.10 Correlation Analysis

Correlation measures the relationship between variables.

#### Types:
- Positive correlation  
- Negative correlation  
- No correlation  

#### Importance:
- Helps in feature selection  
- Useful in predictive modeling  

---

### 4.11 Importance of Statistical Visualization

- Helps understand complex data  
- Identifies patterns and trends  
- Detects outliers and anomalies  
- Improves decision-making  
- Useful in machine learning and data science  

---

## 5. Conclusion

In this experiment, various statistical and specialized visualization techniques such as area plots, pie charts, donut charts, box plots, heatmaps, and bubble plots were studied and applied.

These visualizations helped in understanding data distribution, identifying relationships between variables, and detecting outliers. Advanced plots like heatmaps and bubble plots provided deeper insights into the dataset.

Overall, this experiment demonstrated the importance of statistical visualization in exploratory data analysis and showed how Python tools can effectively represent complex data in an understandable and visually appealing way.
